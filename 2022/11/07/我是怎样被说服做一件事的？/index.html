我是怎样被说服做一件事的？

Captain
2022-11-07

哎，我是怎样被说服做一件事的？

今日晨约5时醒，冲澡早饭后逐渐清醒，开启晨读；当翻开书本前就意识到，又是由“急于求成”的想法导致了周日浑浑噩噩的一天，没有实质的输入和实践，便急于成效反馈，故预期不达焦虑不安
稍读几章便顿释焦虑，好奇为什么这些文章如此有效，并且我是如何被说服的去接受作者的观点；静心分析一番，发现作者每个观点后都列举出一项项做这件事的好处，而且多从作者自己经验出发实际落地的实践；按这些思维方式及行为去探索实践，带来的好处是切实符合自己利益述求，故而自然并无抵触欣然接受

哦，那么如何才能说服一个人去做成某件事呢？

那么如何才能说服一个人去做成某件事呢？或许这其中的核心部分便是私利
信息需要传递，也需要沟通交流传递观点，使目标达成，要多人合作，达成某个目标，那不可避免就要在意见不一时求同存异，采用说服的方式，那么说服有哪些场景

  • 不说服，目标达不成；或者这部分活自己干了
  • 说服，仅用技巧或情侣间甜言蜜语达成自身目的，不涉及对方私利；当下或许接受了，醒悟时刻倍感受欺骗
  • 说服，用私利不用委婉技巧；一句“都是为你好”，往往家长教育子女虽说经验丰富建议中肯，但不容易接受且被抵触
  • 说服，私利+技巧，达成目的

战国策《触龙说赵太后》是个很好的例子，秦急攻赵，赵氏求救于齐，曰必以长安君为质。长安君为赵威后的小儿子，爱子心切，岂肯为质异国受苦；触龙的进言很巧妙，先是言自身年老，行步饮食关切争取同情；再是荐少子补黑衣之数，以卫王宫，博得爱怜少子共鸣；又是以爱燕后贤于长安君巧设悬念烘云托月，从而最后点出“位尊无功，奉厚无劳，—旦山陵崩，难以自托”的困境，彻底的说服了赵威后；前面铺垫为了点出最后的私利部分，便是今令质齐有功于国，而得自立于赵

说服常常用于同等沟通，上对下沟通往往交待便是，下对上沟通说明往往需要考虑周全。如臣子对主上进言，韩非的《说难》一文提及了诸多场景很值得细思-太深刻了，佩服（怎么“舔”出层次）

  • 追求名高说以厚利 -> 进言者被视为节操低下，并受到卑贱待遇，必然会遭人抛弃、被人疏远
  • 追求厚利说以名高 -> 进言者显得没有心计， 而且脱离实际，其建议也必然不会被采纳
  • 阴厚利显名高，说以名高 -> 会在表面上采纳进言者的意见，而在实际上疏远进言者
  • 阴厚利显名高，说以厚利 -> 会在实际上采纳进言者的主张，却在表面上疏远进言者
  • 知饰所说之所矜而灭其所耻 -> 懂得粉饰他自夸的事而掩盖自耻的事
  • 有私急也，必以公义示而强之 -> 有私人的急事，就指明是合乎公义的鼓励去做
  • 其意有下也，然而不能已，说者因为之饰其美，而少其不为也 -> 有卑下念头不能克制，那就粉饰成美好抱怨不去干
  • 心有高也，而实不能及，说者为之举其过而见其恶，而多其不行也 -> 有过高的企求实际不能达到，就举出此事的缺点揭示坏处，称赞他不去做
  • 欲矜以智能，则为之举异事之同类者，多为之地，使之资说于我，而佯不知也以资其智 -> 想自夸才智，就为其多举别的事情中同类之事，多给他提供根据，让他从我这里借用说法，而我却假装不知道帮助他自夸才智
  • 欲内相存之言，则必以美名明之，而微见其合于私利也 -> 想进献与人相安的话，就必须用好的名义阐明它，并暗示他符合其私利
  • 欲陈危害之事，则显其毁诽而微见其合于私患也 -> 想要陈述危害的事，就名言此事会遭到的毁谤，并暗示对他也有害处
  • 誉异人与同行者，规异事与同计者 -> 称赞另一个与他行为相同的人，规划另一件与他考虑相同的事
  • 有与同污者，则必以大饰其无伤也 -> 有和他污行相同的，就必须对它大加粉饰说它没有害处
  • 有与同败者，则必以明饰其无失也 -> 有和他败迹相同的，就必须对它明言掩锦，说他没有过失
  • 毋以其难概之，无以其谪怒之，毋以其败穷之 -> 不用为难的事压抑他，不用过失的事激怒他，不用败绩去困窘他
  • 二人说者皆当矣，厚者为戮，薄者见疑，则非知之难也，处知则难也 -> 武公伐胡，富人失窃，说的直白被杀，说的间接委婉被猜疑，是处理所了解的情况困难
  • 弥子之行未变于初也，而以前之所以见贤而后获罪者，爱憎之变也。故有爱于主，则智当而加亲；有憎于主，则智不当见罪而加疏 -> 宠爱时受亲近；弥子瑕色衰爱弛受憎恶时，才智就显得不恰当，遭到谴责而更被疏远

其实不单单臣子对主公沟通，其实现实生活中弱势对强势，正如感情中处于低位的一样，试图仅靠真诚相待获取信任往往事与愿违，或许唯有察其爱憎兼慎言，旷日离久周泽既渥后方能平等交流；当然也有planB，提升自己或者换个人

私利到底指的是什么呢？
那便是有没有创造价值；给上级创造工作价值，给朋友或恋人创造情绪价值，从而便可影响他人协助自己达成目的；故恰恰说明对会哭的孩子有糖吃存在偏见，会哭的孩子只是让父母有了急切被需要的价值，从而获得所需

另外随记两点

两点不相关的想法也记录下来
1、荀子的慎言

故不问而告谓之傲,问一而告二谓之囋

慎言，急躁和啰嗦都不可取。很有道理，如果是一篇博文内容多，他人大可跳过不读，如果是现实交谈，话多既浪费彼此时间，又可能让他人觉得你啰嗦显摆，或者给其带来被轻视感觉

2、梁惠王的反思

东败于齐,长子死焉;西丧地于秦七百里;南辱于楚

敢于反思正视自身的人物值得学习，反观自身，造遇困窘时常有掩饰逃避想法念头，实不可取，可以从反思日记着手实施；by the way，梁惠王弃商鞅不用也可以理解，毕竟像公司大了采用激进的方式变革转舵风险太大
